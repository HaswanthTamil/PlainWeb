BASE_URL (Local): http://localhost:5001/plainweb-backend/us-central1/api

POST /audit
- Runs a Lighthouse accessibility audit on the provided URL.
- Request Body: { "url": "https://example.com" }
- Response: Processed report JSON including accessibility issues, fix buckets, and Gemini summary.

GET /audit/filtered?url=...
- Returns the filtered Lighthouse LHR JSON.

GET /audit/issues?url=...
- Returns WCAG-enriched accessibility issues.

GET /audit/buckets?url=...
- Returns accessibility issues grouped into fix buckets.

GET /audit/analysis?url=...
- Returns a human-readable accessibility assessment.

GET /audit/summary?url=...
- Returns a friendly summary for website owners (generated by Gemini).
